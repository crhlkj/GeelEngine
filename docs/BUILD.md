## **Инструкция по сборке и интеграции библиотек в проект Geel Engine**

### Общие требования
*Для успешной компиляции проекта требуются следующие компоненты*:

- Современный C++ компилятор, поддерживающий стандарт **C++23** (рекомендуются GCC ≥ 9, Clang ≥ 10, MSVC ≥ VS2019).
- Система сборки **CMake ≥ 3.16**.
- Для рендеринга: совместимая реализация OpenGL (обычно ≥ 3.3 core profile).
- (Опционально) Предварительно собранные исходные коды библиотек **GLFW** и **Dear ImGui** в директории `libs/`.
